# OpsClaw

React + TypeScript + Vite skeleton for the OpsClaw frontend.

## Commands

```bash
pnpm install
pnpm dev
pnpm build
pnpm lint
pnpm typecheck
```

## Structure

```text
src/
  app/            # app shell and router
  components/     # reusable UI building blocks
  features/       # feature-oriented modules
  routes/         # route-level pages
  styles/         # global theme and layout
```
